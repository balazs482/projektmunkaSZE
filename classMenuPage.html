◆ moveArrow()

void MenuPage::moveArrow ( bool  upDirection)
inlinevirtual

Used to move the arrow and execute relevant actions. Movement through the menu items work circularly.

Parameters
upDirectionDirection of the movement. True for moving up and false for moving down.

< Rewrites the line where the arrow has been displayed, without the arrow. With well working screens, only the affacted part could be rewritten.

< If the first option has been selected and the arrow moves up, the last option will be selected.

< If the last option has been selected and the arrow moves down, the first option will be selected.

< If no circular step happens, the neighbouring option in the proper direction will be selected.

< Rewrites the line where the arrow should be displayed, with the arrow. With well working screens, only the affacted part could be rewritten.
